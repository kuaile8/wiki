# 关于

## Edgeless Wiki

Edgeless Wiki 是一个基于 VitePress 构建的开源协作文档平台。

### 设计理念

- **内容为王**：以 Markdown 为核心，让创作者专注于内容本身
- **开放协作**：任何人都可以通过 Fork + PR 参与贡献
- **自动部署**：GitHub Actions 自动构建和发布，减少运维成本
- **灵活扩展**：支持自定义主题、插件和 OAuth 集成

### 技术栈

- [VitePress](https://vitepress.dev/) - 静态站点生成器
- [GitHub Pages](https://pages.github.com/) - 托管服务
- [GitHub Actions](https://github.com/features/actions) - CI/CD
- [Cloudflare](https://www.cloudflare.com/) - CDN 加速（可选）
