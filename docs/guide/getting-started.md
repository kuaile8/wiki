# 快速开始

欢迎使用 Edgeless Wiki！本文档将帮助你快速上手。

## 前置要求

- Node.js 18+ 
- npm 或 yarn

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建生产版本
npm run docs:build

# 预览构建结果
npm run docs:preview
```

## 贡献内容

1. **Fork** 本仓库到你的 GitHub 账户
2. **克隆** 你的 Fork 到本地
3. 在 `docs/` 目录下编辑或创建 Markdown 文件
4. **提交** 更改并推送到你的 Fork
5. 创建 **Pull Request** 到主仓库

## 文档格式

所有文档使用 Markdown 格式编写，支持 VitePress 的所有特性：

-  frontmatter 配置
-  代码块高亮
-  数学公式
-  自定义组件
